## рҹ«Ұ what even IS this

**Gatework** is a browser-based digital logic circuit simulator built with zero frameworks, zero dependencies, and zero excuses. Drop gates, wire them up, simulate signals, get truth tables, boolean expressions, SOP/POS canonical forms вҖ” AND convert between number systems вҖ” all in one dark-mode, neon-green, monospace-font tool that looks like it came out of a hacker's second monitor.

No install. No npm. No webpack config that breaks at 2am. Just open `index.html` and go вҖ” or hit the [live link](https://ShyamDadhania14.github.io/gatework) and use it instantly in your browser.
