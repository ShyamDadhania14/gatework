## рҹ«Ұ what even IS this

**Gatework** is a browser-based digital logic circuit simulator built with zero frameworks, zero dependencies, and zero excuses. Drop gates, wire them up, simulate signals, get truth tables, boolean expressions, SOP/POS canonical forms, AND convert between number systems вҖ” all in one dark-mode, neon-green, monospace-font tool that looks like it came out of a hacker's second monitor.

No install. No npm. No webpack config that breaks at 2am. Just open `index.html` and go.

```
open index.html   # that's it. you're done. ship it.
```

## рҹ“җ Circuit Building

### рҹ”Ө From an Expression

Type any boolean expression into the sidebar:

```
A AND B OR NOT C
(A + B) В· В¬C
A XOR B XNOR C
NOT (A NAND B) OR C
```

Hit **Build Circuit** and the gates auto-layout themselves with wires already connected.

---

### рҹ“Ҡ From SOP / POS Terms

```
Step 1 вҶ’ Choose number of variables (2, 3, or 4)
Step 2 вҶ’ Give them names: A, B, CLK, DATA, whatever
Step 3 вҶ’ Click minterms (SOP) or maxterms (POS) in the grid
Step 4 вҶ’ Hit "Build Circuit"

Result: Full canonical gate network, wired and ready to simulate
```

**SOP** produces AND trees feeding an OR gate.
**POS** produces OR trees feeding an AND gate.
Both auto-insert NOT gates for complemented literals.

---

```

AND + NOT вҶ’  "Behaves like: NAND"
XOR + NOT вҶ’  "Behaves like: XNOR"
3-input majority gate recognized вң“
Hamming parity patterns recognized вң“
```

## рҹ§© How the Circuit Engine Works

```
User places nodes  вҶ’  nodes{}  (id, type, x, y, value, numIns)
User draws wires   вҶ’  wires[]  (from: {node, pin}, to: {node, pin})

Simulate():
  for each node (topological order via recursion):
    computeValue(nodeId):
      if INPUT  вҶ’ return toggle state
      if gate   вҶ’ gather input values from connected wires
                вҶ’ evaluate gate logic
                вҶ’ return result
  update all LEDs, wire colours, gate expression labels
```

Signal propagation is recursive with cycle detection (visited set). MUX and DEMUX handle multi-output simulation via `outPin` parameter.

## рҹӨҜ Things That Go Hard

- **Live expressions on every gate** вҖ” AND gate connected to A, B, C shows `AВ·BВ·C` right on the node body as you wire it
- **BigInt arithmetic** вҖ” the calculator uses JavaScript `BigInt` so `2^64` doesn't turn into `1.8446744e+19`
- **One-click circuit from SOP** вҖ” select minterms 1, 3, 5, 7 вҶ’ full circuit with AND/OR/NOT tree appears instantly
- **Identify any combination** вҖ” the engine brute-forces the truth table and matches against 30+ known patterns including majority gates and inhibition functions
- **IEEE 754 color coding** вҖ” each bit group (sign/exponent/mantissa) has its own color with real exponent, bias breakdown, and NaN/Infinity detection
