How to Use

• Click gate to drop it on board
• Drag to reposition
• Click output pin (right ●) then input pin (left ●) to wire
• Click a wire to delete it
• Double-click INPUT to rename it
• Select gate → use sidebar to change input count (2–6)
• Type expression → Build Circuit to auto-generate
